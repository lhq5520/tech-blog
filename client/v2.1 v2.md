## Phase 13: API Caching

**What is API Caching?**

```
Without cache:
User opens Home → Request API → Wait → Display
User refreshes Home → Request API → Wait → Display (repeated request)

With cache:
User opens Home → Request API → Wait → Display → Store in cache
User refreshes Home → Read from cache → Display immediately (fast!)
```

| Step | Task                       | Description                    |
| ---- | -------------------------- | ------------------------------ |
| 13.1 | Frontend caching           | Use React Query or SWR library |
| 13.2 | Backend caching (optional) | Redis cache for hot data       |
